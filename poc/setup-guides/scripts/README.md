# scripts — baseline POC scripts (no DLP)

These scripts were written for the initial Kafka connectivity test — producer/consumer without DLP tokenization (steps 01–09 of the setup guide). They validate basic Kafka + Schema Registry connectivity before DLP is layered on.

See [`scripts-dlp/`](../scripts-dlp/) for the DLP-enabled versions, and [`streamshield/`](../../../streamshield/) for the production SDK.
